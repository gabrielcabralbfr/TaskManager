# TaskManager
Web Application to manage tasks. Academic project.

## Functionalities
- LocalStorage Authentication
- Create, edit, finish and remove Tasks
- Dashboard module

## Built with
```
- HTML5
- CSS/Bootstrap
- JS/JQuery
```
